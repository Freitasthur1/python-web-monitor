# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [2.0.0] - 2024-12-16

### Adicionado
- Interface web completamente redesenhada com sidebar navegável
- Sistema de notificações por email (SMTP)
- Dashboard com métricas visuais em tempo real
- Formulário completo de configuração de email na interface
- Estrutura de projeto modular e organizada
- API REST para teste de email
- Contador de tempo de atividade (uptime)
- Novo design moderno com gradientes e animações
- Documentação reorganizada em pasta `docs/`
- Scripts utilitários em pasta `scripts/`
- Sistema de configuração em pasta `config/`
- Entry point unificado (`run.py`)
- Script de inicialização melhorado (`start.sh`)

### Melhorado
- Organização do código em módulos separados (src/)
- Interface responsiva com design mobile-first
- Sistema de logs com código de cores aprimorado
- Tratamento de erros mais robusto
- Documentação completa e profissional
- Performance da interface com atualizações otimizadas

### Removido
- Emojis de todos os arquivos (conforme solicitado)
- Código legado e duplicado
- Arquivos desnecessários na raiz

### Correções
- Problemas com paths relativos
- Imports corrigidos para nova estrutura
- Configuração de SMTP mais robusta

## [1.0.0] - 2024-12-15

### Inicial
- Monitoramento básico de editais
- Interface web simples
- Detecção por palavras-chave
- Sistema de logs
- Alertas visuais
